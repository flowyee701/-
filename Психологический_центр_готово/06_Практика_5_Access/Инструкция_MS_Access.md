# Практика 5. База данных обращений

В папке уже лежит готовая рабочая база данных:

- `База_данных_обращений.sqlite`
- `База_данных_обращений.db`

Это один и тот же файл базы в двух распространенных расширениях. Внутри созданы таблицы, первичные ключи, внешние ключи и готовый запрос `Запрос_консультации`.

## Что есть в готовой базе

Таблица `Клиенты`:

- `код_клиента` - первичный ключ
- `фамилия`
- `адрес`

Таблица `Специалисты`:

- `код_специалиста` - первичный ключ
- `фамилия`
- `специализация`

Таблица `Консультации`:

- `код_консультации` - первичный ключ
- `дата`
- `код_клиента` - внешний ключ на таблицу `Клиенты`
- `код_специалиста` - внешний ключ на таблицу `Специалисты`
- `результат`

Связи:

- `Клиенты.код_клиента` -> `Консультации.код_клиента`
- `Специалисты.код_специалиста` -> `Консультации.код_специалиста`

Готовый запрос `Запрос_консультации` выводит поля:

- дата консультации
- клиент
- специалист
- специализация
- результат

## Готовые файлы отчета

Для проверки результата дополнительно подготовлены:

- `Итоговый_отчет_обо_всех_консультациях.csv`
- `Итоговый_отчет_обо_всех_консультациях.html`
- `Проверка_готовой_БД.txt`

HTML-отчет можно открыть в браузере и распечатать или сохранить как PDF.

## Если преподаватель требует именно MS Access

На macOS полноценный файл `.accdb` без установленного Microsoft Access надежно не создается. Поэтому база подготовлена как готовая рабочая база SQLite и как Access-ready пакет CSV. Если нужен именно интерфейс Access, сделай перенос на Windows в Microsoft Access:

1. Открой Microsoft Access.
2. Создай пустую базу данных с названием `База_данных_обращений.accdb`.
3. На вкладке `Внешние данные` выбери импорт из текстового файла.
4. Импортируй `Клиенты.csv`, `Специалисты.csv`, `Консультации.csv` как отдельные таблицы.
5. При импорте отметь, что первая строка содержит имена полей.
6. Для таблицы `Клиенты` назначь первичный ключ `код_клиента`.
7. Для таблицы `Специалисты` назначь первичный ключ `код_специалиста`.
8. Для таблицы `Консультации` назначь первичный ключ `код_консультации`.
9. Открой `Работа с базами данных` -> `Схема данных`.
10. Добавь все три таблицы.
11. Соедини `Клиенты.код_клиента` с `Консультации.код_клиента`.
12. Соедини `Специалисты.код_специалиста` с `Консультации.код_специалиста`.
13. В обоих случаях включи `Обеспечение целостности данных`.
14. Создай новый запрос в режиме SQL и вставь текст из файла `Запрос_консультации.sql`.
15. Сохрани запрос под названием `Запрос_консультации`.
16. Создай форму на основе таблицы `Консультации`.
17. Оставь в форме поля: дата, код клиента, код специалиста, результат.
18. Создай отчет на основе запроса `Запрос_консультации`.
19. Назови отчет `Итоговый отчет обо всех консультациях`.

После этого Access-файл будет полностью повторять подготовленную здесь базу.
