# Практика 5. База данных обращений в MS Access

## Таблицы
Импортируйте CSV-файлы в Access как три отдельные таблицы:
- `Клиенты`: первичный ключ `код_клиента`
- `Специалисты`: первичный ключ `код_специалиста`
- `Консультации`: первичный ключ `код_консультации`

## Схема данных
Создайте связи:
- `Клиенты.код_клиента` -> `Консультации.код_клиента`
- `Специалисты.код_специалиста` -> `Консультации.код_специалиста`

Включите обеспечение целостности данных.

## Запрос
Создайте запрос в режиме SQL и вставьте содержимое файла `Запрос_консультации.sql`.
Итоговые поля должны быть: дата консультации, клиент, специалист, специализация, результат.

## Форма ввода
Создайте форму на основе таблицы `Консультации`.
Разместите поля: дата, код клиента, код специалиста, результат.
Для кодов клиента и специалиста можно использовать поля со списком.

## Отчет
Создайте отчет на основе запроса `Запрос_консультации`.
Название отчета: `Итоговый отчет обо всех консультациях`.
