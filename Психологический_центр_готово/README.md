# Готовый комплект работ

Папки пронумерованы по заданиям. В каждой папке лежат файлы для сдачи или переноса в нужный сервис.

- `01_Лабораторная_1_Сетевое_анкетирование`: отчет, структуры Google Forms, таблицы ответов.
- `02_Практика_1_АРМ_психолога`: DOCX и PDF со структурной схемой.
- `03_Практика_2_Шаблон_Word`: форма приема клиента.
- `04_Практика_3_Excel`: книга Excel по шести заданиям.
- `05_Практика_4_PowerPoint`: презентация отчета психолога.
- `06_Практика_5_Access`: готовая SQLite/DB база, CSV, SQL, итоговый отчет и инструкция для переноса в MS Access.
- `07_Практика_6_Сайт`: локальный многостраничный сайт.
- `08_Практика_7_Анкетирование`: тест, инструкция и шаблон ответов.
- `09_Инструкции_и_чеклисты`: ручные действия перед сдачей.
