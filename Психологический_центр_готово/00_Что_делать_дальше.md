# Что делать дальше

В папке `Психологический_центр_готово` лежат готовые файлы по всем работам. Папка `tools` к сдаче не относится.

## Что уже готово

- Отчеты Word по лабораторной 1, практике 1, практике 2 и практике 7.
- Excel-файлы с расчетами, ответами анкет и структурой вопросов.
- PowerPoint-презентация отчета психолога.
- CSV, SQL и инструкция для базы обращений в MS Access.
- Многостраничный сайт психологического центра.
- PDF-схема АРМ психолога.

## Что нужно заменить своими данными

- `[ФИО]` - во всех титульных листах и презентации.
- `[Группа]` - во всех титульных листах и презентации.
- Ссылка на собственную анкету уже вставлена в отчет лабораторной 1; ссылка на отдельный тест тревожности нужна только если преподаватель требует реальную форму для практики 7.

## Что нужно предоставить, если преподаватель требует реальные данные

- Свою ФИО и группу.
- Ссылку на опубликованную Google Form.
- Файл ответов Google Forms после минимум 3 прохождений.
- Ссылку на опубликованный сайт.
- Скриншоты опубликованного сайта и формы.
- Реальные фотографии специалистов, если нельзя использовать учебные изображения-заглушки.
- Файл `.accdb`, если преподаватель требует именно готовую базу Access, а не CSV/SQL для импорта.

## Порядок действий

1. Открой Word/PPTX и замени `[ФИО]`, `[Группа]`.
2. Создай Google Form по файлам структуры в папках лабораторной 1 и практики 7.
3. Получи минимум 3 ответа и экспортируй таблицу.
4. При необходимости опубликуй сайт из папки `07_Практика_6_Сайт`.
5. В MS Access импортируй CSV-файлы из папки `06_Практика_5_Access` и создай связи, запрос, форму, отчет.
6. Перед отправкой используй архив `Психологический_центр_готово.zip`.
