# Что делать дальше

В папке `Психологический_центр_готово` лежат готовые файлы по всем работам. Папка `tools` к сдаче не относится.

## Что уже готово

- Отчеты Word по лабораторной 1, практике 1, практике 2 и практике 7.
- Excel-файлы с расчетами по всем 10 заданиям практики 3, ответами анкет и структурой вопросов.
- PowerPoint-презентация отчета психолога.
- Готовая база обращений `База_данных_обращений.sqlite` / `База_данных_обращений.db`, CSV, SQL, итоговый отчет и инструкция для MS Access.
- Многостраничный сайт психологического центра.
- PDF-схема АРМ психолога.
- Реальные выгрузки ответов Google Forms: 4 ответа по собственной анкете и 3 ответа по тесту тревожности.

## Что нужно заменить своими данными

- `[ФИО]` - во всех титульных листах и презентации.
- `[Группа]` - во всех титульных листах и презентации.
- Ссылка на собственную анкету уже вставлена в отчет лабораторной 1; ссылка на отдельный тест тревожности нужна только форма для практики 7 уже добавлена.

## Что нужно предоставить, если преподаватель требует реальные данные

- Свою ФИО и группу.
- Ссылки на опубликованные Google Forms уже добавлены.
- Файлы ответов Google Forms уже добавлены в папки лабораторной 1 и практики 7.
- Ссылку на опубликованный сайт.
- Скриншоты опубликованного сайта и формы.
- Реальные фотографии специалистов, если нельзя использовать учебные изображения-заглушки.
- Файл `.accdb`, только если преподаватель требует именно нативный файл Microsoft Access. Готовая рабочая база уже есть в формате SQLite/DB.

## Порядок действий

1. Открой Word/PPTX и замени `[ФИО]`, `[Группа]`.
2. Проверь, что опубликованные Google Forms открываются по ссылкам из отчетов.
3. Перед сдачей открой таблицы ответов и убедись, что реальные выгрузки подходят преподавателю по числу респондентов.
4. При необходимости опубликуй сайт из папки `07_Практика_6_Сайт`.
5. Для практики 5 можно сдавать готовую базу `База_данных_обращений.sqlite` / `База_данных_обращений.db` и отчет. Если нужен именно MS Access, импортируй CSV-файлы из папки `06_Практика_5_Access` и создай связи, запрос, форму, отчет по инструкции.
6. Перед отправкой используй архив `Психологический_центр_готово.zip`.
